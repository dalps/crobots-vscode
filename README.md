# crobots-vscode

This Visual Studio Code extension provides IntelliSense for the CROBOTS programming language.

## Features

* Document symbols
* Syntax Highlighting
* Go-to definition
* References view
* Inline completions
* Variable renaming for keywords, local variables and intrinsic functions
* Intrinsic function documentation

## Contributing

The language features are powered by a [chevrotain](https://chevrotain.io/docs/) parser written in TypeScript.

Issues and pull requests are welcome.
